# Thunderbolt

**Thunderbolt is a multipurpose discord bot which aims to provide almost all features within one bot.**

## Prefix

`tb! [configurable]`

## Commands

```

I am lazy to write it xD
```

```
- Moderation

- Memes
- Fun
- Games
- Economy
- Automod
- ServerStats
- Welcomer
- Logging
- Autorole
- Configurable
- AFK
- And Many More
```
